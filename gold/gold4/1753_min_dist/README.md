가중치가 달라지면 bfs가 꺠진다.
bfs은 모든 간선의 가중치가 같을 때만 최단 거리를 올바르게 구한다. 
그래서 먼저 들어온 경로가 길면 짧은 것보다 먼저 처리돼서 오답이 된다. 
이 문제는 가중치가 있으니 bfs를 쓰면 안됨
